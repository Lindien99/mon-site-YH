Recherche web universelle — vue par défaut Affiche tous les types de résultats : pages web, extraits enrichis, Knowledge Graph, cartes, actualités, images, vidéos… Classés par pertinence.
🖼️ Images ↗
Galerie d'images indexées sur le web Affichage en grille avec filtres :
  • Taille (petite / moyenne / grande)
  • Couleur dominante
  • Type (photo, clipart, dessin, GIF)

📰 Actualités ↗
Articles de presse récents Résultats issus de sources journalistiques vérifiées, triés par date. Filtres disponibles : période (24 h, semaine, mois), pays, langue.
🎬 Vidéos ↗
Vidéos longues (YouTube, Dailymotion…) Documentaires, reportages, visites virtuelles, émissions TV. Filtres : durée, date, qualité HD, sous-titres, source.
📱 Vidéos courtes ↗
Shorts / Reels / TikToks indexés Contenus verticaux de moins de 3 minutes : YouTube Shorts, Instagram Reels, TikTok. Format rapide, idéal pour un aperçu visuel d'un lieu.
